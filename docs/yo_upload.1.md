# NAME

```sh
yo upload [options]
```

# DESCRIPTION

Upload kernel source

# OPTIONS

*--assume-yes*
:   Assume yes to all questions.

*--dry-run*
:   Dry run

*--no-testing*
:   Don't upload verification branches

Part of the **yo(1)** suite
